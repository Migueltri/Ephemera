# Ephemera
El proyecto es un sistema de ciberseguridad para controlar la huella digital. Identifica datos personales expuestos y reduce su visibilidad (ocultación o desindexación) manteniendo la trazabilidad. Integra un módulo de rastreo que detecta reapariciones o usos indebidos, logrando una gestión estructurada y verificable de la exposición online.
