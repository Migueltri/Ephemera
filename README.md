# Ephemera
Este proyecto consiste en el desarrollo de un sistema de ciberseguridad orientado al control de la huella digital y la exposición de datos personales en entornos online. La solución permite identificar elementos asociados a la identidad digital de una persona (nombres, perfiles, imágenes o referencias) y modificar su nivel de visibilidad mediante procesos de ocultación o desindexación operativa, sin eliminar la trazabilidad de la información. El sistema incorpora un módulo de registro de actividad y seguimiento de incidencias que detecta posibles reapariciones o usos indebidos del contenido tratado. De este modo, se plantea una infraestructura que transforma la gestión de la exposición digital en un proceso controlado, verificable y estructurado dentro del ámbito de la ciberseguridad.
